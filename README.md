
# FutureCode Case


## Installation

Clone the repo
```bash
https://github.com/dilancirkin/futurecode-case.git
```

Install NPM packages

```bash
  npm install
```

Run the app
```bash
  npm  start
```

  
## Technologies

[React JS](https://react.dev/)

[Bootstrap](https://getbootstrap.com/)

[React Router](https://reactrouter.com/en/main)

[FontAwesome](https://fontawesome.com/)
